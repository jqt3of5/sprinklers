Welcome to my RESTful gpio service! This service provides an interface to the gpio pins on a Raspberry Pi.

Important Endpoints:

PUT /gpio/:pin


Configure a pin to be input/output, and allocate it to the current user

DELETE /gpio/:pin


Release the gpio pin from the current user

GET /gpio/:pin


Get the current value of a pin. Mostly used for pin configured as input pins.

POST /gpio/:pin/clr


Set the value of an output pin to low

POST /gpio/:pin/set


set the value of an output pin to high
